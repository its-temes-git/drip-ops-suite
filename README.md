# Sawkem Fashion — Streetwear Ops

Internal management platform for Sawkem Fashion, Summit Branch, Addis Ababa.

## Features
- Public storefront with product catalog
- Owner dashboard with inventory & analytics
- Sales portal for staff
- JWT-based authentication

## Tech Stack
- **Frontend**: React + TypeScript + Vite + TailwindCSS
- **Backend**: Node.js + Express
- **Database**: PostgreSQL (Supabase)
